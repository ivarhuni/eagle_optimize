# Optimization Context Skill

When working on personal optimization questions in this repository:

1. Read `profiles/current_profile.md` first if it exists.
2. Check `profiles/assessments/` for newer or conflicting onboarding snapshots.
3. Search `optimizations/` for prior reports that overlap with the current topic.
4. Prefer information grounded in those files over generic assumptions.
5. Call out missing context when the stored notes are insufficient.
6. If the topic involves prescription drugs, controlled substances, diagnoses, or dosing:
   - do not act like a prescriber
   - summarize the relevant user context
   - surface risks, uncertainties, and clinician questions
   - avoid direct instructions to change dose or treatment

Use the category folders consistently:

- `optimizations/foodLiquids`
- `optimizations/drugsSupplements`
- `optimizations/mentalHeadspace`
- `optimizations/body`
- `optimizations/dailyRoutine`
